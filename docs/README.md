# Nuxt 3, Notion API, Pinia, TDD, Auth App

## Start a new Nuxt 3 App

```sh
nuxi init [App Name]
```


## Modules Installed

```sh
# DEV Dependencies
yarn add -D @pinia/nuxt   # For State management
yarn add -D vitest        # For unit testing
yarn add -D c8            # For code coverage (Part of unit testing)



```
